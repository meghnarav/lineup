# LineUp — Landing Page (Demo)

## Live deployment:

Open [https://koncertify-lineup.vercel.app/](https://koncertify-lineup.vercel.app/)

---

This is a demo Next.js landing page for LineUp — a fair concert ticket queue system.

---

## Quick start

### 1. Install dependencies:

```bash
npm install
```

### 2. Run the development server:

```bash
npm run dev
```

Open: 
```bash
http://localhost:3000
```

---

## Notes
- Built with Next.js (App Router), Tailwind CSS, and Framer Motion.
- The "Join the Queue" button assigns a random queue position and simulates countdown.
